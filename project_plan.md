# Mide Cakes and Crafts - Project Plan

## 1. Project Description
A beautiful, warm, and inviting website for Mide Cakes and Crafts - a home-based cake and crafts business in Chelmsford, Essex. The website showcases custom cakes, cupcakes, and handmade crafts to attract local customers and drive sales through an easy order form.

**Target Users:** Local customers in Chelmsford/Essex area looking for custom cakes, cupcakes, and handmade crafts for events, celebrations, and gifts.

**Core Value:** Showcase beautiful products, provide clear pricing, make ordering simple and accessible.

## 2. Page Structure
- `/` - Home (Hero, featured products, about snippet, testimonials)
- `/about` - About Us (Owner story, business values, location)
- `/products` - Products (Cakes, Cupcakes, Crafts categories)
- `/pricing` - Pricing (Clear price list for all offerings)
- `/gallery` - Gallery (Photo showcase of past work)
- `/contact` - Contact (Contact info, map, message form)
- `/order` - Order Form (Custom order submission form)

## 3. Core Features
- [ ] Beautiful hero section with business branding
- [ ] Product showcase with categories
- [ ] Pricing table with clear rates
- [ ] Photo gallery of past work
- [ ] Contact form and business details
- [ ] Custom order form (cake type, size, flavor, date, contact)
- [ ] Responsive design for mobile and desktop
- [ ] Smooth animations and transitions

## 4. Data Model Design
No database needed for Phase 1. Order form will use Readdy's built-in form submission system.

## 5. Backend / Third-party Integration Plan
- **Supabase:** Not needed for initial phase (no user accounts, no database storage)
- **Shopify:** Not needed (custom orders, not standard products)
- **Stripe:** Not needed (customer handles payment separately)
- **Forms:** Use Readdy form submission for order form

## 6. Development Phase Plan

### Phase 1: Core Website Build
- **Goal:** Build all 7 pages with complete content, styling, and functionality
- **Deliverable:** Fully functional multi-page website with navigation, all sections, order form, and responsive design
- **Pages:** Home, About, Products, Pricing, Gallery, Contact, Order Form

### Phase 2: Polish & Animations
- **Goal:** Add scroll animations, hover effects, image optimization, and final visual polish
- **Deliverable:** Production-ready website with smooth interactions and professional feel
